1. 为什么做“索引”

近年来，中国在全球治理与国际组织体系中的参与程度持续提升，越来越多国际组织、国际合作平台及相关附属机构开始在中国设立总部、代表处、办事处、研究中心或合作平台。与此同时，北京、上海、杭州、海南等地陆续出台政策，推动国际组织及相关国际功能性机构落户，并逐步形成国际组织集聚发展的趋势。然而，中国（大陆）现行规范并未对“位于中国的国际组织”作出统一定义，而是依据设立基础、登记路径和机构属性进行分类规制：境外非政府组织按《境外非政府组织境内活动管理法》登记或备案[1]，社会团体、基金会等分别纳入国内法人制度框架[2][3]，地方政策则常从业务领域和功能集聚角度理解“国际组织”[4][5]。在统计层面，国际协会联盟《国际组织年鉴》、公安部境外非政府组织办事服务平台、民政部全国社会组织查询平台和联合国驻华系统分别提供了不同口径的信息来源[6][7][8][9]。上述集合总体互补，也存在交叉和覆盖不足，尤其较难完整呈现联合国等国际组织与国内机构共建的附属机构、依托机构、联合中心或实验室。本索引基于上述公开渠道，对位于中国的国际组织进行调研、整理与收录，以尝试实现对当前位于中国的国际组织基本信息的分类统计与直观呈现。

2. 统计口径

本“索引”中的“中国国际组织”并不严格采用国际政府间与非政府间组织的传统定义，而是综合国际组织的定义、使命与功能行动、以及对公开信息的研判，采用功能性和本土化的视角，将中国语境下的“国际组织”定义为具有总部、秘书处、代表处、办事处、中心、研究机构、合作平台、基地、行动网络、常设国际会议论坛等相关功能性实体或规律性活动，并具有明显或较强国际组织、国际合作或跨境公共事务属性的机构。

当前版本的“索引”对机构的收录口径如下：

  1. 联合国系统在华机构。包括联合国机构驻华代表处、办公室等独立实体；
  2. 自主成立的国际性组织。包括国际政府间和非政府间组织。其中，国际政府间组织主要通过官方公开信息证实并收录，如上海合作组织；国际非政府间组织主要为在“全国社会组织查询平台”登记，且状态为“正常”，在机构属性和实际行动中体现出强国际组织性质的社会组织；
  3. 在“境外非政府组织办事服务平台”登记，且状态为“正常”的境外非政府组织；
  4. 《国际组织年鉴》数据库收录，具有强国际组织性质，且经过网络公开信息证实其具有实际运营实体、载体或活动的机构（见注A1）；
  5. 本土化的“国际组织”。这部分机构包括：
    • 挂靠在高校、研究所或事业单位等国内机构的国际组织，如教科文组织二类中心；
    • 自主设立或由国际组织和国内机构合作建立，属性为事业单位或社会组织，或与事业单位等机构“一个机构、两块牌子”，但扮演强国际组织功能的机构，如国际竹藤中心；
    • 国际组织与地方机构合作建立的，拥有非国际组织属性或弱实体性的，以工作团队的形式进行活动的项目组、基地、平台、行动网络和实验室。

注：

  1. 注A1 本“索引”收录过程中对机构进行了运营实体、虚拟载体和活动的调研。具体方式包括通过网络公开信息，证明机构拥有实际的办公地点、官方网站、微信公众号、学术期刊、活动页面，或者有涉及机构的官方的新闻报道等公开信息。以上条件满足其一则进行收录。不代表收录机构一定处于绝对活跃，或持续开展活动的状态。
  2. 注A2 本“索引”的收录最小单位为项目组。如某一机构在不同城市拥有代表处、办公室或项目组，则分别统计为不同机构。此外，如果机构总部和常设执行机构/秘书处的定位或功能有一定差异，则也统计为不同机构，如国际小水电联合会和国际小水电中心。
  3. 注A3 本“索引”对香港特别行政区、澳门特别行政区与台湾省的机构的统计口径暂为：
    • 位于香港特别行政区、澳门特别行政区的机构，暂仅纳入联合国系统驻地办公室，和具备一定行业影响力的国际政府间组织和非政府间组织，当前版本暂仅收录以下机构：香港国际仲裁中心、国际调解院、国际移民组织驻香港办公室、海牙国际私法会议亚太区域办事处、联合国大学-岭南大学人道创新及科技中心、联合国大学驻澳门研究所、国际商会仲裁院秘书处亚洲事务办公室、国际统一私法协会亚太地区联络办公室、国际火锅产业联盟、世界宠物协会、澳门质量品牌国际认证联盟、全球林产品绿色供应链（澳门）联合会、上海国际仲裁（香港）中心，以及上述机构在其他城市的代表处/办事处。
    • 当前版本暂未纳入位于台湾省的机构。
    • 位于港澳台，且公益性质或企业属性占主导的机构，部分虽在“境外非政府组织办事服务平台”登记，但当前版本暂未纳入。
  4. 注A4 本“索引”仅基于公开信息，基于中国本土视角进行信息调研与统计，不等同于法律意义上对“国际组织”的资格认定，也不构成对机构法律地位、主管关系或代表资格的正式确认。
  5. 注A5 本“索引”所有收录机构的信息均为调研统计网络官方公开信息所得，仅供参考。本索引名录不代表任何收录机构的观点和立场，相关信息请以机构官方发布途径为准。
  6. 注A6 本“索引”的当前版本暂未纳入ISO, IEC, IEEE等国际标准组织的国内官方技术对口单位和委员会，暂未纳入部分非实体化的国际机制。
  7. 注A7 本“索引”将不定期纳入新成立或进行组织调整的机构。具体请参考本页标题下方的版本号。

3. “索引”有什么

3.1 “机构列表”页

本页包括对各收录机构的中文名、外文名、机构来源、行动领域、设立年份、所在地、官网和微信公众号基本信息的呈现，提供卡片和列表视图。用户可以通过检索机构的中文名和外文名获取机构信息，并按照机构来源、行动领域、设立年份和所在地进行筛选。

3.1.1 中文名

  • 各机构中文名均采用各机构网络公开信息。
  • 其中，境外非政府组织的中文名统一按照境外非政府组织办事服务平台的命名规范[7]，国际性社会组织的中文名统一按照全国社会组织查询平台的命名规范[8]。
  • 根据公开可证实信息源，对总部、秘书处、代表处、办事处等不同机构定位进行标注。

3.1.2 外文名

  • 各机构外文名均采用各机构网络公开信息。
  • 其中，英文名能够公开证实的机构，采用机构英文名；机构英文名不能公开证实但其他语种命名能够证实的，采用其他语种命名，机构定位后缀采用英文；所有外文名均不能公开证实的，该条目为空缺。
  • 所有机构默认不加定冠词前缀，官网机构单独命名、Logo等独立标识中有定冠词的除外。
  • 所有英文简称能够公开证实的机构，均附英文简称。

3.1.3 机构来源

  • 本筛选条目根据机构的登记与官网机构收录属性，在当前版本设置境外非政府组织、国际性社会组织、联合国系统和国际组织年鉴条目。
  • 带有境外非政府组织标签的，代表该机构在境外非政府组织办事服务平台登记[7]，且状态为“正常”。
  • 带有国际性社会组织标签的，代表该机构在全国社会组织查询平台的登记[8]，且状态为“正常”。
  • 带有联合国系统标签的，代表该机构为联合国机构驻华代表处、办公室和工作组；同时，本索引对“联合国系统”采取广义标签口径，将联合国机构与中国地方单位共建的中心、研究所、实验室、行动网络等合作机构也纳入该标签。
  • 带有国际组织年鉴标签的，代表该机构被国际组织年鉴收录[6]，且经查询证实具有国际组织属性。
  • 上述标签会根据机构的状态动态变化，最新信息请以上述各官方来源为准。

3.1.4 行动领域

  • 本筛选条目根据机构的功能属性对各机构做简单功能划分。

3.1.5 设立年份

  • 本筛选条目为各机构办公室设立的年份，并非严格的机构成立年份。
  • 对各机构设立的年份判定的优先采用各机构官网公示的设立年份，其次为网络官方公开报道该机构设立的年份。
  • 对于无法查询到设立年份公开信息的境外非政府组织，采用该机构在境外非政府组织办事服务平台[7]的登记的年份为设立年份。
  • 对于根据国际协定设立的机构，设立年份优先采用正式协定生效时间，其次为该机构实际营业时间。
  • 对于将总部搬迁到中国的机构，设立年份采用总部正式迁入的时间，而非该机构自身成立的时间。

3.1.6 所在地

  • 本筛选条目为各机构办公室所在地，若单一机构在中国有不同地点的办公室，可以分别查询到。
  • 当前版本所在地具体到地级市。对于设立在县级市的部分机构，暂列入该县级市所属或代管的地级市。

3.1.7 官网

  • 本筛选条目为各机构官方网站，优先收录该机构中国代表处/办事处的中文网站，其次为主管或委托单位的网站、总部的中文网站、英文网站和其他外文网站。
  • 官网不能公开证实的，该条目为空缺。

3.1.8 微信公众号

  • 本筛选条目为各机构微信公众号，优先收录机构创建的认证的微信公众号，其次为主管或委托单位的微信公众号，个别为长期发布该机构相关动态信息的行业微信公众号，不代表该机构官方性质，仅供了解该机构动态信息。微信公众号不能公开证实的，该条目为空缺。

3.2 “中国国际组织地图”页

本页包括对各收录机构分布的直观呈现。用户可以点击地图中的各城市获取该城市的国际组织信息，并通过检索和筛选（详见上一节的介绍）查看分布。

  • 本页采用中华人民共和国标准地图（审图号：GS(2023)2767号）。
  • 当前版本所在地具体到地级市。对于设立在县级市的部分机构，暂列入该县级市所属或代管的地级市。

3.3 “使用说明”页

本页包括中国国际组织索引的介绍、收录标准和使用方法。

4. 如何使用“索引”

在“机构列表”页，用户可以查看当前数据库收录的机构数量，并通过任何关键词检索机构，查看具体机构的信息。用户亦可按照机构来源、行动领域、设立年份和所在地进行单选或多选。此外，用户可通过切换卡片和列表视图切换浏览偏好，并且在列表视图点击表头的排序方式进行顺序浏览。

在“中国国际组织地图”页，用户可以点击各城市查看该城市拥有的机构。用户亦可以通过检索和条件筛选查看相同机构来源、行动领域和设立年份机构的分布情况。此外，用户在点击城市后也可以在城市内部进行检索和筛选。

在“使用说明”页，用户可以了解中国国际组织索引的介绍、收录标准和使用方法。
